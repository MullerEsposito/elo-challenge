
# Elo Challenge




## Run Locally



Clone the project

```bash
  git clone https://github.com/MullerEsposito/elo-challenge
```

Go to the project directory

```bash
  cd elo-challenge
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Running Tests

To run tests, run the following command

```bash
  yarn cypress:open
```
